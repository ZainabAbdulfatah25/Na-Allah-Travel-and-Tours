# Na-Allah Travels and Tours

A modern, responsive travel agency website built with React, TypeScript, and Vite.

## Features
- **Immersive Hero Section**: Featuring high-quality imagery and a search widget.
- **Destination Packages**: Displaying Hajj, Umrah, and holiday packages with specific details.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop.
- **Custom UI**:
  - **Polished Typography**: Using 'Outfit' and 'Inter' fonts.
  - **Themed Colors**: Deep Ocean Blue (`#020617`) and Vibrant Cyan accents.
  - **3D Social Icons**: Custom CSS-styled glossy buttons for social media.

## Technologies
- React
- TypeScript
- Vite
- CSS Modules

## Getting Started

1. Clone the repository
2. Run `npm install`
3. Run `npm run dev` to start the local server.
